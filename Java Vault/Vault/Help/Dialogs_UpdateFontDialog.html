Update Font Dialog

Use this dialog box to change the outline and default text font, foreground color, and background color. Note: The outline background color can only be changed on Linux systems.

Dialog Box Fields and Buttons:
Outline Font and Colors...
Press this button to change the outline's font, foreground color, and background color. Note: The outline's background color can only be changed on Linux systems.
Default Text and Colors...
Press this button to change the text window's font, foreground color, and background color.
