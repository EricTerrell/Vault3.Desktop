License Terms Dialog

Use this dialog box to decide if you accept the Vault 3 program and source code license terms.

Dialog Box Fields and Buttons:
License Terms
Read the license terms.
I reject these terms
Select this radio button if the license terms are not agreeable to you.
I accept these terms
Select this radio button if the license terms are agreeable to you.
OK
Press this button to close this dialog box.
