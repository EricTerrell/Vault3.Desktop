Encryption

When you choose to password-protect a Vault 3 document, the file is securely encrypted using the AES algorithm, with an 128-bit key. For best security, password-protect all Vault 3 documents that contain private information.

Note: be sure to memorize all passwords, or record them in a secure place. If your password is lost or forgotten, it will not be possible to decrypt the file.
