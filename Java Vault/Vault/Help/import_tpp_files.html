How to Import The Photo Program Files into Vault 3

You can easily convert your The Photo Program files to Vault 3 files:

  1. You will need the latest, registered version of The Photo Program. If you are a registered user, request the latest version of The Photo Program.
  2. Run The Photo Program and open the file you want to convert to Vault 3 format:
  3. Select File / Export:
  4. Specify the export filename and specify XML Files (*.xml) as the type.
  5. Press the save button.
  6. Go to Vault 3 and select File / Import Vault or The Photo Program XML File:
  7. Navigate to the file you exported from The Photo Program and click Open:
  8. Expand the outline item by clicking the +:
  9. Select all items except for the first:
  10. Select Outline / Unindent:
  11. Select the first outline item. Then select Outline / Remove:
