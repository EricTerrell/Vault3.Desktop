Outline / Import Pictures

Use this command to add pictures to your Vault 3 document. You can add the following kinds of picture files to Vault 3:

  • JPEG (*.jpg)
  • GIF (*.gif)
  • Windows Bitmaps (*.bmp)
  • Portable Network Graphics (*.png)
  • Tagged Image Format (TIFF) Files (*.tif)
  • Windows Metafiles (*.wmf)
Select one or more files and press Open. The pictures will be added at the current position in the outline.

Shortcuts:
Toolbar:
Keys:Ctrl+I
