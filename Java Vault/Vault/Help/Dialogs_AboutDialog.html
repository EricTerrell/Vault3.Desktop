Help / About Dialog

Use this dialog box to view copyright information, information about contributors, license terms, and to support Vault 3 development.

Dialog Box Fields and Buttons:
Donate
Press this button to support Vault 3 development with a donation via PayPal.
Read License Terms
Press this button to read the license terms for Vault 3 and its source code.
Send Feedback about Vault 3
Press this button to ask questions about Vault 3, or request Vault 3 enhancements.
Check for Updates
Press this button to find out if there is a more recent version of Vault 3.
Vault 3 for Android
Vault 3 is available for smart phones and tablets. Press this button for information on the Android version of Vault 3.
