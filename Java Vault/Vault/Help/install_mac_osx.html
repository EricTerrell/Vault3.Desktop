How to Install Vault 3 on Mac OS X

  1. Download the 32 or 64-bit version of Vault 3 for Mac OS X.
  2. Unzip the downloaded file to /Applications/Vault 3/Vault3Cocoa_x86_32 (32-bit) or /Applications/Vault 3/Vault3Cocoa_x86_64 (64-bit)
  3. Create an Automator workflow to launch Vault 3. Specify the appropriate command:

    32-bit java -d32 -XstartOnFirstThread -jar "/Applications/Vault 3/Vault3Cocoa_x86_32/vault_mac.jar" &
    64-bit java -XstartOnFirstThread -jar "/Applications/Vault 3/Vault3Cocoa_x86_64/vault_mac.jar" &
