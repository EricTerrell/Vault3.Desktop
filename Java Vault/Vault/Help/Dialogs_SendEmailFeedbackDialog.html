File / Send Feedback Dialog

Use this dialog box to send feedback about Vault 3 to Eric Bergman-Terrell.

Dialog Box Fields and Buttons:
From:
Enter your email address.
Subject:
Enter the subject of the email.
Body:
This is the body of the email which you can optionally edit before sending.
Send
Press this button to send the email.
Cancel
Press this button to close the dialog box without sending email.
