How to Install Vault 3 on Ubuntu (Linux)

Step 1: Install the Java Runtime Environment

  1. Select Applications / Add/Remove.
  2. Drop down the Show drop-down. Choose All available applications.
  3. Enter Sun Java in the Search text box.
  4. Check the check box to the left of Sun Java 6 Runtime in the list of Applications.
  5. Click Apply Changes.
  6. Click Apply.

Step 2: Install Vault 3

  1. Download Vault 3 for 32-bit Linux (GTK) (e.g. Ubuntu) or Vault 3 for 64-bit Linux (GTK) (e.g. Ubuntu).
  2. Unzip the file to a hard disc folder.
  3. Right-click the Desktop.
  4. Select Create Launcher.
  5. Enter Vault 3 in the Name text box.
  6. Enter the following in the Command text box: java -jar {folder into which you extracted the .zip file}/vault_linux_gtk.jar
  7. Click Close.

Step 3: Run Vault 3

Double-click the Vault 3 launcher.
