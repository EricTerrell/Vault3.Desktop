Outline / Paste Dialog

Use this dialog box to specify if the pasted outline will be pasted above or below the first outline item.

Dialog Box Fields and Buttons:
Above first item
Select this radio button to paste the item above the first outline item.
Below first item
Select this radio button to paste the item immediately after the first outline item.
OK
Press this button to save your changes.
Cancel
Press this button to discard your changes.
