How to Import Vault Files into Vault 3

You can easily convert your Vault files to Vault 3 files:

  1. Run Vault and open the file you want to convert to Vault 3 format:
  2. Select Options / Settings, go to the Misc tab and make sure the "Number outline items in printouts, e-mails, and exports" check box is cleared.
  3. Select Outline / Collapse All:
  4. Select all outline items:
  5. Select File / Export:
  6. Specify the export filename and specify XML Files (*.xml) as the type.
  7. Press the save button.
  8. Go to Vault 3 and select File / Import Vault or The Photo Program XML File:
  9. Navigate to the file you exported from Vault and click Open:
  10. Expand the outline item by clicking the +:
  11. Select all items except for the first:
  12. Select Outline / Unindent:
  13. Select the first outline item. Then select Outline / Remove:
